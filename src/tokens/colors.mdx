# Colors

## Primaries

### Contextual Primary colors

Primary colors react to the context that is set in `DesignProvider`
Wherever possible use `primary` and set the context in the DesignProvider that wrappes that part of the app.

<ColorSwatches colors={colors} context="primary" />

### Shadower Colors

These colors are assigned to `primary` when `context='shadower'` on the `DesignProvider`

<ColorSwatches colors={colors} context="shadower" />

### Host Colors

These colors are assigned to `primary` when `context='host'` on the `DesignProvider`

<ColorSwatches colors={colors} context="host" />

## Neutral colors

Use `neutral` for body text, backgrounds, etc.

<ColorSwatches colors={colors} context="neutral" />

## Whites

Use `whiteout` for backgrounds. They have a very subtle gold tint to contrast with the blue hue of the neutral.

<ColorSwatches colors={colors} context="whiteout" />

## State Palettes

The following colors are to be used very selectively to highlight a specific state. Errors, warnings, etc.
