# Banner

Banner is a [Flex](/components/Flex) and is to be used to create a large banner with a background image.

## Props

<Props of={Banner} />
Props table might not render due to a [bug in docz](https://github.com/pedronauck/docz/issues/777)

### Main Props:

- `backgroundImage`: maps to CSS attribute `background-image: url()` location of the background image
- `backgroundSize`: maps to CSS attribute `background-size`, defaults to `cover`
- `backgroundPosition`: maps to CSS attribute `background-position`, defaults to `center center`
- `backgroundRepeat`: maps to CSS attribute `background-repeat`, defaults to `no-repeat`
- `shadow`: origin/dark side of the shadow gradient [see below](#shadow). Defaults to `bottom`
- `children`: use normal react children as the children. Often you'll want a `<Container>` as your only child to contain text within.

## Banner as the site title

Use this setup for large page-headers.

<Playground>
  <Banner backgroundImage="http://placekitten.com/1001/700">
    <Container>
      <Heading as="h1">The main site title</Heading>
      <Lede>A little intro to why users should read on</Lede>
    </Container>
  </Banner>
</Playground>

## Different height / No picture

Overwrite the default `height` or set a `bg` color rather than an image.

<Playground>
  <Banner bg="primary.dark" height="50vh">
    <Container>
      <Heading as="h2" color="white">
        Some smaller title, maybe
      </Heading>
    </Container>
  </Banner>
</Playground>

## Shadow

You can change the anchor of the shadow. Default is to `bottom`

- `bottom`: shadow is dark at the bottom, light at the top
- `right`: shadow is dark at the right, light at the left
- `top`: shadow is dark at the top, light at the bottom
- `left`: shadow is dark at the left, light at the right
- `none`: no shadow at all

<Playground>
  <Banner bg="primary.dark" height="50vh" shadow="none">
    <Container>
      <Heading as="h2" color="white">
        Some smaller title, maybe
      </Heading>
    </Container>
  </Banner>
</Playground>
