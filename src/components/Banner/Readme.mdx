### Main Props:

- `backgroundImage` - maps to CSS attribute `backgroundImage` location of the background image
- `backgroundSize`: maps to CSS attribute `backgroundSize`, defaults to `cover`
- `backgroundPosition`: maps to CSS attribute `backgroundPosition`, defaults to `center center`
- `backgroundRepeat`: maps to CSS attribute `backgroundRepeat`, defaults to `no-repeat`
- `children`: use normal react children as the children. often you'll want a `<Container>` as your only child to contain text within.
