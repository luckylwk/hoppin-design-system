# Heading

The Heading is used for all the Headings across the app.

## Props

<Props of={Heading} />
Props table might not render due to a [bug in docz](https://github.com/pedronauck/docz/issues/777)

The main props to set are:

- **as** - tag and level of heading to use. `h1` - `h6` defaults to `h2`

Make sure you only have one h1 per page.

If you need more options, consider using [Text](components/Text) or [Paragraph](components/Paragraph).
