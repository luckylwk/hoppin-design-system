# DesignProvider

The DesignProvider should wrap the main App component, it provides the theme to all the siblings.

Make sure to set `context='host'` or `context='shadower'` in the to set primary colors, it defaults to `context='shadower'`.

- `host` => primary is host pink
- `shadower` => primary is shadower blue
