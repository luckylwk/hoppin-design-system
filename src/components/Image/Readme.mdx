# Image Components

- [Tile](#tile) Image tile

## Tile

This image was specifically designed for the experience cards (3/2 aspect ratio), but can be applied to more aspect ratios now.

### Props

<Props of={Tile} />
Props table might not render due to a [bug in docz](https://github.com/pedronauck/docz/issues/777)

Underneath it uses the [Box](/components/Box) component.

Tile has a
- Required prop for `src` that needs the image URL
- Optional prop `ratio` that can take `'1/1'`, `'3/2'` or `'4/3'` currently. It defaults to `'1/1'`
- Optional prop `borderRadius` that defaults to `'3px'`

The width will adjust to the width of the parent container (100%) and then determine the height accordingly.

  </Box>
</Playground>

## Avatar

Component created specifically for rendering an avatar. Underneath it uses the Box and [Tile](#tile) components

### Props

<Props of={Avatar} />
Props table might not render due to a [bug in docz](https://github.com/pedronauck/docz/issues/777)

Underneath it uses the [Box](/components/Box) component.

Avatar has a
- Required prop for `src` that needs the image URL
- Optional prop `size` that defaults to `'base'`
- Optional prop `squared` flag that defaults to `false`
