# Image Components

- [Tile](#tile) Image tile
- [Avatar](#avatar) Basic avatar

## Tile

This image was specifically designed for the experience cards (3/2 aspect ratio), but can be applied to more aspect ratios now.

### Props

<Props of={Tile} />
Props table might not render due to a [bug in docz](https://github.com/pedronauck/docz/issues/777)

Underneath it uses the [Box](/components/Box) component.

Tile has a
- Required prop for `src` that needs the image URL
- Optional prop `ratio` that can take `'1/1'`, `'3/2'` or `'4/3'` currently. It defaults to `'1/1'`
- Optional prop `borderRadius` that defaults to `'3px'`

The width will adjust to the width of the parent container (100%) and then determine the height accordingly.

<Playground>
  {() => {
    const src = 'https://storage.googleapis.com/hoppin-platform-avatars/usr-5bd7570e081c0e3b0d036917/1563490774170000_avatar_600x600.jpeg';
    const card = 'https://storage.googleapis.com/hoppin-platform-images/br-5b44c9ad96738038d071ffc9/exp/1569549455741000_card_600x400.jpeg';
    return (
      <Box>
        <Box width="200px" padding="base" bg="primary.lighter">
          <Tile src={src} />
        </Box>
        <Box width="300px" marginTop="large" padding="base" bg="primary.lighter">
          <Tile src={card} ratio="3/2" />
        </Box>
        <Box width="500px" marginTop="large" padding="base" bg="primary.lighter">
          <Tile src={card} ratio="4/3" borderRadius="20px" />
        </Box>
      </Box>
    );
  }}
</Playground>

## Avatar

Component created specifically for rendering an avatar. Underneath it uses the Box and [Tile](#tile) components

### Props

<Props of={Avatar} />
Props table might not render due to a [bug in docz](https://github.com/pedronauck/docz/issues/777)

Underneath it uses the [Box](/components/Box) component.

Avatar has a
- Required prop for `src` that needs the image URL
- Optional prop `size` that defaults to `'base'`
- Optional prop `squared` flag that defaults to `false`

<Playground>
  {() => {
    const src = 'https://storage.googleapis.com/hoppin-platform-avatars/usr-5bd7570e081c0e3b0d036917/1563490774170000_avatar_600x600.jpeg';
    return (
      <Box width="400px" height="150px" padding="base" bg="primary.lighter">
        <Avatar src={src} paddingRight="base" />
        <Avatar src={src} paddingRight="small" squared />
        <Avatar src={src} size="icon" alignSelf="flex-start" />
      </Box>
    );
  }}
</Playground>
