# Expandable

The Expandable is a collection of utility components that work together to create expandable/togglable content.

It doesn't style the expandable content, that's left up to you when you build the layout.

Make sure `Expandable` has exactly one instance of `ExpandableToggle` and one instance of `ExpandableBody`

## Props

<Props of={Expandable} />

<Props of={ExpandableBody} />

For an example how to use `toggleDisplay` on `ExpandableBody` to disable automatic toggling of `display: block` to `display: none` when using animations, check out [Navigation](/comoponents/Navigation)

## Many Toggles, initial state

Since context is held locally, there can be many Expandables on the same page. You should even be able to nest them.

Set prop `initExpanded={true}` to initialise the component in an extended state.

<Playground>
  <Expandable onToggle={(expanded)=> { console.log(expanded ? 'expanded': 'closed')}}>
    <ExpandableToggle>
      <Heading as='h4'>Click to expand</Heading>
    </ExpandableToggle>
    <ExpandableBody>
      <Paragraph>And you can read so much more. It's a miracle</Paragraph>
    </ExpandableBody>
  </Expandable>
<Expandable initExpanded={true}>
  <ExpandableToggle>
    <Heading as="h4">Starting expanded</Heading>
  </ExpandableToggle>
  <ExpandableBody>
    <Paragraph>And you can read so much more. It's a miracle</Paragraph>
  </ExpandableBody>
</Expandable>
<Expandable>
  <ExpandableToggle>
    <Heading as="h4">Click to expand</Heading>
  </ExpandableToggle>
  <ExpandableBody>
    <Paragraph>And you can read so much more. It's a miracle</Paragraph>
  </ExpandableBody>
</Expandable>
</Playground>

## Indicate expanded/collapsed state

`Expandable` exports ExpandableContext, use it to render any icon or other styles you need to indicate expanded/collapsed state.

<Playground>
  {() => {
    // import {Expandable, ExpandableToggle, ExpandableBody, ExpandableContext } from 'hoppin-design-system'
    // import { useContext } from 'react'
    const CustomToggle = () => {
      const { isExpanded } = useContext(ExpandableContext);
      return (
        <Heading as="h4" color={isExpanded ? "primary.base" : "inherit"}>
          {isExpanded ? "Click to close" : "Click to open"}
        </Heading>
      );
    };
    return (<React.Fragment>
        <Expandable>
          <ExpandableToggle>
            <CustomToggle />
          </ExpandableToggle>
          <ExpandableBody>
            <Paragraph>And you can read so much more. It's a miracle</Paragraph>
          </ExpandableBody>
        </Expandable>
        <Expandable>
          <ExpandableToggle>
            <Heading as="h4">Add ExpandableToggleIcon to indicate toggle state <ExpandableToggleIcon /></Heading>
          </ExpandableToggle>
          <ExpandableBody>
            <Paragraph>And you can read so much more. It's a miracle</Paragraph>
          </ExpandableBody>
        </Expandable>
      </React.Fragment>
    );
}}
</Playground>
