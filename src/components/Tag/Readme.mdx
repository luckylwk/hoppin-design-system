# Tag

The Tag is used to list themes or keywords related to an experience or blog post.
It can be used by itself, or in combination with a [Heading](/components/Heading). If used before a Heading, it behaves the same as [Capotion](/components/Caption). It nests itself on top of the Heading to give context.
