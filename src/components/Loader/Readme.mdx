# Loaders

Various types of loaders

A simple "typing" style loader... Use this for generic loading screens, where you can't use [Skeleton](#skeleton)

## Skeleton

<Props of={Skeleton} />

Fantastic to show that actual **content** is being loaded. Use wherever possible.

Skeleton has a

- Optional prop for `ratio` that defaults to `3/2`
- Optional prop for `borderRadius` that defaults to `3px`
- Optional prop for `width` that defaults to `100%`

<Playground>
  <Flex>
    <Box margin="none" padding="base" bg="neutral.light" width="240px">
      <Skeleton ratio="3/2" />
      <Box marginTop="small">
        <Skeleton ratio="title" borderRadius="0" width="90%" />
        <Skeleton marginTop="small" ratio="text" width="75%" />
      </Box>
    </Box>
    <Box margin="none" marginLeft="base" padding="base" width="240px">
      <Skeleton ratio="3/2" />
      <Box marginTop="small">
        <Box display="inline-block" width="26px">
          <Skeleton ratio="1/1" borderRadius="50%" />
        </Box>
        <Skeleton
          display="inline-block"
          marginLeft="small"
          ratio="title"
          borderRadius="3px"
          width="80%"
        />
        <Skeleton marginTop="small" ratio="text" width="75%" />
      </Box>
    </Box>
  </Flex>
</Playground>
