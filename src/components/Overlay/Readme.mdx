# Overlay

Overlay component to rendering an overlay either within an element or covering the entire screen.

## Props

<Props of={Overlay} />

* `position`: defaults to `fixed` for a full page overlay

### Contained overlay

When passing `position="absolute"` it will fit to the parent. This is useful for, for example, blocking a form and showing a loading animation.

<Playground>
  {() => {
    const OverlayContent = () => (
      <Flex width="100%" height="100%" justifyContent="center" alignItems="center">
        <HeartLoader size={40} context="shadower" />
      </Flex>
    )
    return (
      <Box 
        position="relative" 
        width="700px" 
        height="200px"
        bg="neutral.base"
      >
        {/** margin="base" only to show it fits to the parent container */}
        <Overlay position="absolute" margin="base">
          <OverlayContent />
        </Overlay>
      </Box>
    )
  }}
</Playground>

### Page overlay

We use `Expandable` to allow for toggling on and off in this case.

<Playground>
  {() => {
    const OverlayContent = () => (
      <Flex width="100%" height="100%" justifyContent="center" alignItems="center">
        <ExpandableToggle>
          <Button context="neutral">
            <FiX /> Close page overlay
          </Button>
        </ExpandableToggle>
      </Flex>
    );
    return (
      <Expandable>
        <ExpandableToggle>
          <Button context="neutral">Click to show page overlay</Button>
        </ExpandableToggle>
        <ExpandableBody>
          <Overlay>
            <OverlayContent />
          </Overlay>
        </ExpandableBody>
      </Expandable>
    )
  }}
</Playground>
