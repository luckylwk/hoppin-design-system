# Paragraph

The Paragraph is used for body copy. It comes with default margins and is optimised for readability.

## Props

<Props of={Paragraph} />
Props table might not render due to a [bug in docz](https://github.com/pedronauck/docz/issues/777)

The main props to set are:

- **marginBottom** - set to 0, if you don't want the default margin.
- **color** - is set to `inherit` to adapt to the context, but you might want to explicitly set the color if needed.

If you need more options, consider using [Text](components/Text) or [Heading](components/Heading) instead.
