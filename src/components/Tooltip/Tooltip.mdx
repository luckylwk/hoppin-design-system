# Tooltip

The Tooltip is used to show extra information on hover. Use sparingly as it onluy works on desktop, since mobile devices have no hover state.

Tooltip is based on [react-tooltip](https://github.com/wwayne/react-tooltip). Check out the [react-tooltip docs](https://react-tooltip.netlify.com/) for more usage examples.
Don't use the `react-tooltip` prop `type`. Instead use `context` as you would for a [Button](/components/Button)

Use `data-tip` attribute on any element to attach a `Tooltip` to it.
Then add `data-for="someid"` to the same element.
Finally give the `<Tooltip id='someid'>` to link the two.
