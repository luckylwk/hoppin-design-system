# Chat

ChatBubble is a [Box](/components/Box) with a bit of extra styling, it can be used for a list of items, or to highlight an important piece of UI.

Together with ChatMeta it forms the content of a chat conversation.

## Props

<Props of={ChatBubble} />
<Props of={ChatMeta} />

Props table might not render due to a [bug in docz](https://github.com/pedronauck/docz/issues/777)

### Main props for ChatBubble

- `variant`: `me` or `them`
- `avatarURL`: url to avatar image. Currently only used for `variant="them"`
- `children`: string or react elements to fill the chat bubble with

### Main props for ChatMeta

- `children`: string

## Usage

Wrap it in a `Container` and you're good to go.

<Playground>
