# Textarea

The Textarea is used to WYSIWYG text editing. It can be configured to allow styling with a subset of Markdown styles.

The Textarea is based on [GetOutline.com's open source editor](https://github.com/outline/rich-markdown-editor), which itself is based on [slate.js](https://github.com/ianstormtaylor/slate).

It outputs a simple MD string, that can be parsed using the Markdown component or transformed with the [unified.js](https://unified.js.org/) family of tools.
