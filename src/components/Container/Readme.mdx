# Container

Container is a [Flex](/components/Flex) has a max-width and centers content on the page, it defaults to `display: flex`.

## Props

<Props of={Container} />
Props table might not render due to a [bug in docz](https://github.com/pedronauck/docz/issues/777)

Set `width="narrow"` for to optimize body text width, set it to `base` for a wider layout, `full` to fill any available space.

Only set `maxWidth` if absolutely necessary and you cant use narrow/base widths.

Container implements the styled-system props for:

- [space](https://styled-system.com/api#space) (only use full-length props. e.g. `paddingBottom`, not `pb`)
- [layout](https://styled-system.com/api#layout)
- [color](https://styled-system.com/api#color)
- [border](https://styled-system.com/api#border)
- [flexbox](https://styled-system.com/api#flex)
