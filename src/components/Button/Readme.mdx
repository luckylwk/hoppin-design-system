# Button

Buttons are components which signify that an action will occur when a user interacts with them.

<Props of={Button} />

## Standard Button

This is your standard button. They typically trigger one action when interacted with, providing the basis for creating an interactive page. Add additional context to them using the context prop.

<Playground>
  <ButtonGroup>
    <Button>Click me</Button>
    <Button context="shadower">Shadower</Button>
    <Button context="host">Host</Button>
    <Button context="danger">Danger</Button>
  </ButtonGroup>
  <Box bg="primary.base" padding="large" marginTop="large">
    <Button context="whiteout">Whiteout</Button>
  </Box>
</Playground>

## Outline Button

These are our outline buttons. Use them for secondary actions. Most of the time, this will be when used alongside a standard button or primary action.

<Playground>
<ButtonGroup>

<Button variant='outline'>Click me</Button>
<Button variant='outline' context="shadower">Shadower</Button>
<Button variant='outline' context="host">Host</Button>
<Button variant='outline' context="danger">Danger</Button>
</ButtonGroup>
<Box bg="primary.base" padding="large" marginTop="large">
  <Button variant="outline" context="whiteout">Whiteout</Button>
</Box>
</Playground>

## Subtle Button

These are our subtle buttons. Use them for when the action is not overly important and should blend into its surroundings.

<Playground>
<ButtonGroup>

<Button variant='subtle'>Click me</Button>
<Button variant='subtle' context="shadower">Shadower</Button>
<Button variant='subtle' context="host">Host</Button>
<Button variant='subtle' context="danger">Danger</Button>

</ButtonGroup>

<Box bg="primary.base" padding="large" marginTop="large">
  <Button variant="subtle" context="whiteout">Whiteout</Button>
</Box>
</Playground>

## Sizes

All variants can come in three sizes.

    </Button>
    <Button display="inline-block" size="large">

## Icon Buttons

Buttons that contain _just_ an icon automatically adjust padding to become round.

## Groups of Buttons

Use `<ButtonGroup>` to automatically space out Buttons.

`ButtonGroup` is based on `Flex` and takes all flex props. It defaults to `flex-direction: row` and will automatically apply a `small` space between buttons.

<Playground>
  <ButtonGroup>
    <Button>Click me</Button>
    <Button>Click me</Button>
    <Button>Click me</Button>
    <Button>Click me</Button>
  </ButtonGroup>
</Playground>

Change the layout to be vertical, by applying `flexDirection='column'`

<Playground>
  <ButtonGroup flexDirection="column">
    <Button>Click me</Button>
    <Button>Click me</Button>
    <Button>Click me</Button>
    <Button>Click me</Button>
  </ButtonGroup>
</Playground>
