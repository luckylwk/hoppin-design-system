# Card

Card is a [Box](/components/Box) with a bit of extra styling, it can be used for a list of items, or to highlight an important piece of UI.

Box implements the styled-system props for:

- [space](https://styled-system.com/api#space) (only use full-length props. e.g. `paddingBottom`, not `pb`)
- [layout](https://styled-system.com/api#layout)
- [color](https://styled-system.com/api#color)
- [border](https://styled-system.com/api#border)
- [flexbox](https://styled-system.com/api#flex)

It add a prop for `elevation` that controls the size of the drop shadow. _A number between 0 an 5 - defaults to 3/base'_
